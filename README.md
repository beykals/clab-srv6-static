# Лабораторная работа: Настройка IPv6 и SRv6

## Описание

В данной лабораторной работе настраивается базовая IPv6-сеть и конфигурация **SRv6 (Segment Routing over IPv6)** для построения **IPv4-связности поверх IPv6 underlay-сети** с использованием **SRv6 инкапсуляции**.

Основная цель — обеспечить передачу IPv4-трафика через IPv6-инфраструктуру, используя механизмы SRv6 c SRH в режиме TE.

---

## Адресация

### Loopback-интерфейсы

| Протокол | Формат адреса |
|-----------|----------------|
| IPv4      | `172.16.1.{ROUTER_NODE}/32` |
| IPv6      | `2001:db8::{ROUTER_NODE}/128` |

### Линковые соединения

| Протокол | Формат адреса |
|-----------|----------------|
| IPv6      | `2001:db8:0:{ROUTER_NODE+NEXT_ROUTER_NODE}::/127` |

### Локаторы SRv6

| Назначение | Формат |
|-------------|---------|
| SRv6 Locator | `5f00:0:{ROUTER_NODE}:0:{FUNC}::/64` |
