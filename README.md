В лабе настраивается базовая настройка ipv6, конфигурация srv6, а также построение ipv4 связности поверх ipv6 underlay сети с использованием srv6 инкапсуляции.

Адресация
Loopback
ipv4: 172.16.1.x/24
ipv6: 2001:db8::x/64
Линковые соединения
ipv6: 2001:db8:x:0::/127
Локаторы srv6
fc00:x::/64
